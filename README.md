open V2Ray Key Tester (это бета версия программа будет доробатываться и пока она доступна только для linux mint)

тестер V2Ray ключей с полной поддержкой всех протоколов

Поддержка всех протоколов:

* **VMess** - классический протокол V2Ray
* **VLESS** - современный легкий протокол
* **VLESS + Reality** - новейший стелс-протокол
* **Trojan** - имитация HTTPS трафика
* **Shadowsocks / SS2022** - быстрый и простой
* **Hysteria2** - QUIC-based, очень быстрый
* **TUIC** - также на QUIC
* **SSH** - туннелирование через SSH

### QR коды:

* Генерация QR для любого ключа
* Сканирование QR с экрана
* Сканирование QR из файла
* Сохранение QR в изображения

### Система подписок (Subscriptions):

* Добавление неограниченного количества подписок
* Автоматическое обновление
* Статистика по каждой подписке
* Группировка серверов по подпискам

### Расширенная фильтрация:

* Фильтр по протоколу
* Фильтр по группам
* Фильтр по статусу (работает/не работает)
* Живой поиск по всем полям
* Множественные фильтры одновременно

### Улучшенное управление:

* Копирование выбранных ключей
* Удаление дубликатов
* Удаление нерабочих серверов
* Избранное (favorite)
* Редактирование ключей
* Экспорт в различные форматы

### Продвинутый интерфейс:

* Контекстное меню (правая кнопка)
* Горячие клавиши
* Цветовая индикация (зеленый/желтый/красный)
* 4 вкладки (Ключи, Статистика, Лучшие, Графики)
* Детальная информация по каждому серверу
* Автовыбор лучшего сервера

## Установка

### Шаг 1: Установка системных зависимостей

**Linux Mint / Ubuntu / Debian:**

```bash
sudo apt update
sudo apt install python3 python3-pip python3-tk -y
```

### Шаг 2: Установка Python библиотек

```bash
pip3 install --break-system-packages -r requirements_advanced.txt
```

**Или через виртуальное окружение:**

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements_advanced.txt
```

### Шаг 3: Запуск программы

```bash
chmod +x v2ray_tester_advanced.py
python3 v2ray_tester_advanced.py
```

## Быстрый старт

### 1. Добавление ключей

**Способ 1: Из буфера обмена**

1. Скопируй V2Ray ключи (vmess://, vless://, trojan://, ss:// и т.д.)
2. В программе: `Ctrl+V` или кнопка "Добавить"

**Способ 2: Из файла**

1. Создай файл .txt с ключами (каждый на новой строке)
2. В программе: `Ctrl+O` или кнопка "Загрузить"

**Способ 3: Через подписки**

1. Меню → Подписки → Управление подписками
2. Добавь URL подписки
3. Нажми "Обновить выбранную"

**Способ 4: Сканирование QR кодов**

1. Меню → Инструменты → Сканировать QR
2. Выбери файл с QR кодом или скан с экрана

### 2. Тестирование

**Быстрое тестирование всех:**

* `Ctrl+T` или кнопка "Тест всех"

**Тест выбранных:**

* Выдели ключи → Правая кнопка → "Тест сервера"

**Автоматический мониторинг:**

* Настрой интервал в верхней панели
* Нажми "Старт мониторинг"

### 3. Работа с ключами

**Копирование:**

* Выдели ключи → `Ctrl+C` или Правая кнопка → "Копировать ключ"

**Генерация QR:**

* Правая кнопка на ключе → "Генерировать QR"
* Или пакетная генерация через меню инструментов

**Экспорт:**

* Меню → Файл → Экспорт выбранных/всех
* Сохрани в .txt файл

**Удаление:**

* Выдели ключи → `Delete` или Правая кнопка → "Удалить"
* Меню → Правка → Удалить дубликаты/нерабочие

### 4. Фильтрация и поиск

**Фильтры:**

* Протокол: выбери нужный протокол
* Группа: фильтр по группам/подпискам
* Статус: работает/не работает/не тестировано

**Поиск:**

* Введи текст в поле поиска
* Ищет по названию, серверу, стране

**Сброс фильтров:**

* Кнопка "Сбросить фильтры"

### 5. Просмотр результатов

**Вкладка "Все ключи":**

* Полный список с сортировкой
* Двойной клик → подробная информация
* Правая кнопка → контекстное меню

**Вкладка "Статистика":**

* Общая статистика по всем ключам
* Распределение по протоколам и странам
* Средняя производительность

**Вкладка "Лучшие ключи":**

* ТОП-10 самых быстрых
* ТОП-10 самых стабильных
* ТОП-10 по uptime
* Кнопка "Автовыбор лучшего"

## Подробные инструкции

### Работа с подписками

**Добавление подписки:**

1. Меню → Подписки → Управление подписками
2. Нажми "Добавить подписку"
3. Введи название и URL
4. Нажми "Добавить"

**Обновление подписок:**

* Одной: Выбери подписку → "Обновить выбранную"
* Всех: Меню → Подписки → Обновить все подписки

**Автообновление:**

* Программа может автоматически обновлять подписки при запуске
* Или настрой периодическое обновление

### Генерация QR кодов

**Для одного ключа:**

1. Правая кнопка на ключе → "Генерировать QR"
2. Окно с QR кодом откроется автоматически
3. Можно сохранить изображение

**Для нескольких ключей:**

1. Меню → Инструменты → Генератор QR кодов
2. Выбери нужные ключи
3. Нажми "Генерировать QR"
4. Откроются окна с QR для каждого

### Экспорт и импорт

**Экспорт ключей:**

```
vmess://...
vless://...
trojan://...
ss://...
```

**Экспорт статистики:**

* Вкладка Статистика → "Экспорт статистики"
* Сохраняется в текстовый файл

**Сохранение конфигурации:**

* Меню → Файл → Сохранить конфигурацию
* Сохраняется в JSON с ключами и настройками

### Интерпретация результатов

**Задержка (Latency):**

* < 100 ms - отлично
* 100-300 ms - хорошо
* > 300 ms - медленно

**Успешность:**

* > 90% - очень стабильный
* 70-90% - стабильный
* < 70% - нестабильный

**Статусы:**

* Работает - сервер доступен
* Недоступен - не работает
* Не тестировано - еще не проверяли

### Автовыбор лучшего сервера

**Как работает:**

1. Анализирует все протестированные серверы
2. Учитывает задержку и успешность
3. Выбирает оптимальный вариант

**Использование:**

* Вкладка "Лучшие ключи"
* Кнопка "Автовыбор лучшего"
* Программа предложит скопировать ключ

## Устранение неполадок

### Проблема: QR коды не работают

```bash
pip3 install --break-system-packages qrcode pillow
```

### Проблема: Не сканируются QR коды

```bash
sudo apt install libzbar0
pip3 install --break-system-packages pyzbar
```

### Проблема: Программа не запускается

```bash
python3 --version
python3 -c "import tkinter"
sudo apt install python3-tk
```

### Проблема: Подписки не обновляются

* Проверь интернет-соединение
* Проверь URL подписки (должен начинаться с http:// или https://)
* Некоторые подписки могут быть заблокированы

### Проблема: Неправильная задержка

* Текущая версия использует TCP ping
* Для реального теста через V2Ray нужна интеграция с V2Ray Core
* Планируется в следующей версии
